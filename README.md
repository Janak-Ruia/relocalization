To Run:
1. Clone the repo:
  > git clone https://github.com/Janak-Ruia/relocalization.git

2. Change paths appropriately in pcl_publish.cpp and run.launch 

3. Build using 
  > catkin_make -j1
   and source the workspace


4. Run using:
  > roslaunch relocalization run.launch
